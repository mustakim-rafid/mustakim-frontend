# PPA - Landing page

A high-fidelity, responsive landing page built with a focus on premium aesthetic design. This project implements a structured path to productivity through a modern web interface.

---

## 🚀 Tech Stack

* **Framework:** Next.js
* **Language:** TypeScript
* **Styling:** Tailwind CSS, shadcn/ui, Framer motion
* **Icons:** Lucide React

---

## ✨ Key Features

* **Interactive Curriculum:** A modular accordion-based course breakdown with active lesson states.
* **Dynamic Pricing Section:** Includes a real-time countdown timer, high-end border glow effects, and clear value propositions.
* **Responsive Architecture:** Pixel-perfect execution across mobile, tablet, and desktop breakpoints.
